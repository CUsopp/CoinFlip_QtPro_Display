# CoinFlip_QtPro_Display
My QtProject Here!

## 一款翻金币的小游戏~

### 开始界面 包含背景、标题、开始按钮；同时菜单栏支持退出软件；

### 点击开始按钮，进入 关卡选择界面；同时菜单栏支持退出软件；

### 关卡选择界面 包含返回按钮，点击可返回 开始界面；

### 点击关卡，进入 关卡界面；关卡界面包含返回按钮，可返回 关卡返回界面；包含关卡提示艺术文字；同时菜单栏支持退出软件；

### 关卡界面添加背景；背景上添加硬币；不同关卡，金银币初始情况不同；

### 点击硬币，实现硬币动态翻面；

### 记录硬币位置关系，根据位置关系实现硬币范围翻面；

### 胜利曲线提醒生成；胜利后硬币不可翻面；

### 声音提醒；
